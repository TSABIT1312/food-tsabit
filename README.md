## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Configure Firebase:
   - Update `src/firebase.js` with your Firebase configuration

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
